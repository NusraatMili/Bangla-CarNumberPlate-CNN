## Dataset

This folder should contain 11 subfolders:
০, ১, ২, ৩, ৪, ৫, ৬, ৭, ৮, ৯, dash

Each folder contains at least 30 grayscale images of that digit
cropped from real Bangladeshi car number plates.

Total: Around 330 images across 11 classes.

Dataset is not uploaded due to size.
Collect your own images from real BD car plates
or contact the author via LinkedIn.
